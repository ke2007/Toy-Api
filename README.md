# ⛑ ToyProject- 게시판 Api

## SpringBoot로 게시판 api를 구현한 Toy프로젝트

<hr>

### 프로젝트 개요

> Spring 을 공부한 뒤에 만들어본 게시판 api입니다.

<br>

### 프로젝트 목표

> 게시판에 필요한 기능( 댓글, 추천 비추천, 게시글 조회, 게시글 목록 조회 등 ) 을 구현해 본다.

<br>
